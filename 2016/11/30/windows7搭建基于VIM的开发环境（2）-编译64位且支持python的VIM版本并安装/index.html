下载VIM源码

从 https://github.com/vim/vim 下载vim源码，下载zip包比直接git clone https://github.com/vim/vim.git速度快一些。将下载的源码解压到$VIM_MASTER_PATH。

1.打开msys2 command,切换到$VIM_MASTER_PATH/src

2.复制Make_cyg_ming.mak编译脚本为current.mak

3.编辑current.mak
