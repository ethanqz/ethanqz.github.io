本系列指导 主要是从无到有完整搭建windows下基于VIM的开发环境，主要分为以下几部分：

  • 搭建msys2下64位编译环境
  • 编译64位且支持python的VIM版本并安装
  • VIM基本配置
  • VIM安装vundle
  • VIM安装开发环境所需插件<本篇>

本篇我们主要讲解vim安装开发环境所需插件。其中jedi-vim和syntastic需要vim添加python支持，neocomplete需要vim添加lua支持，在第二篇我们已经完成了vim编译的python和lua的支持。

#安装依赖

  1. 安装pip，pip官方地址下载最新版pip-9.0.1.tar.gz，下载后解压并运行python setup.py install。

  3. 安装pyflakes，pip install pyflakes,syntastic代码检查依赖。

  4. 安装jedi，pip install jedi，jedi-vim代码不全依赖。
